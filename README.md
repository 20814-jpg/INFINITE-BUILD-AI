# 🚀 INFINITE BUILD AI

An AI-powered app builder that generates React applications from natural language prompts.

## Features

- 🧠 AI-powered code generation
- ⚡ Live preview in iframe
- 🎨 Simple web UI
- 📦 Express backend
- 🔄 Real-time updates

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. Clone the repository
```bash
git clone https://github.com/20814-jpg/INFINITE-BUILD-AI.git
cd INFINITE-BUILD-AI
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

1. Type a prompt (e.g., "build todo app")
2. Click the "Build" button
3. Watch your app generate and preview in real-time!

## Current Capabilities

- **Todo App**: Type "todo app" to generate a basic todo application
